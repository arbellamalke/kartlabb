# Vite app with Storybook

I created this repo doing the following:

1. `pnpm create vite vite-react-ts-sb --template react-ts`
2. `cd vite-react-ts-sb`
3. `pnpm install`
4. `npx sb init`
5. `pnpm run build-storybook`

and it took more than 6 minutes.
